# Nationality_Emotion_Detection
In this repository, the gui application detects nationality and emotion of a person along with some specifications based on the detected nationality.

1. To use my Nationality and Emotion Detection Model
2. Just clone or download and unzip my respository
3. Install tensorflow and cv2 in the system using pip
4. Run the gui.py file
5. Upload the image in the gui
6. Click on the Detect button to detect the nationality and emotion.
